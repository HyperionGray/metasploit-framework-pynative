# Legacy Ruby Modules

This directory contains legacy Ruby modules that were moved here during the Python migration.

## Contents

- `exploits/` - Legacy Ruby exploit modules
- `auxiliary/` - Legacy Ruby auxiliary modules  
- `post/` - Legacy Ruby post-exploitation modules
- `payloads/` - Legacy Ruby payload modules
- `encoders/` - Legacy Ruby encoder modules
- `nops/` - Legacy Ruby NOP modules

All modules here are preserved for historical reference but are no longer part of the active codebase.
