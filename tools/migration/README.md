# Ruby to Python Migration Tools

This directory contains tools for migrating Ruby code to Python as part of the post-2020 Python migration initiative.

## Files

- `migrate_ruby_to_python.py` - Main migration script
- `analyze_ruby_files.py` - Ruby file analysis tool
- `verify_migration.py` - Migration verification tool
- `test_*.py` - Test and development scripts
